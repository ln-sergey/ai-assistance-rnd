# Чёрный список фраз и паттернов, которые «выдают» синтетику.
# Используется в scripts/data/synthesize-validate.ts.
#
# Формат:
#   - Каждая непустая строка, не начинающаяся с #, трактуется как
#     JS-регулярка. Матч регистронезависимый, флаги "iu" (i = ignore
#     case, u = Unicode для корректной работы с кириллицей).
#   - Для последовательности букв (включая кириллицу) — \p{L}+.
#   - Пробельные паттерны: \s+ для одного-или-более пробелов и т.п.
#
# Стартовый набор синхронизирован с prompts/synthesize-card-v1.txt
# (раздел 4 Anti-clickbait). Расширяй по мере появления повторяющихся
# маркеров синтетики в реальных прогонах.

# --- Превосходные степени ---
лучший в мире
лучший в россии
лучший в \p{L}+
непревзойд[её]нный
легендарный
самый известный

# --- Анти-уникальность ---
уникальная экскурсия
уникальный маршрут
не имеющий аналогов
аналогов\s+нет
эксклюзивная программа
эксклюзивный тур

# --- Гарантии ---
100\s*%\s*гарантия
полная\s+гарантия
гарантированный\s+восторг

# --- Скрытость / маркеры «инсайдер» ---
секретный маршрут
секретное место
только у нас
никто\s+другой\s+не

# --- Шаблонная рекламная патетика ---
незабываем[аоые]\p{L}*
неповторим[аоые]\p{L}*
волшебн[аоые]\p{L}*\s+(атмосфер|приключени)
